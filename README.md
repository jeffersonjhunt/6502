6502 Playground
===========================================================

This is just a bunch of random sudo projects for working on the MOS 6502 CPU.

## External Refs ##

https://www.applefritter.com/replica/chapter7

## Emulator ##

To run the emulator

```
mkdir build
cd build
cmake ../emulator
make
ln -s ../emulator/logging.cfg
./apps/emulator
```
